Meryl
=====

Meryl is minimalist web framework for nodejs platform.
It is really simple to use, fun to play and easy to modify.

	require('meryl')
		.h('GET /', function (req, resp) { resp.send('<h3>Hello, World!</h3>'); });
		.run();

Meryl has much more...
